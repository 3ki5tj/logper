Cycles of the logistic map

Introduction

The logistic map is defined as:
xk+1 = f (xk) ≡ r xk (1 − xk).
An n-cycle means
x n = x0.
The problem is to determine the r values that allow stable n-cycles.

Stable cycles

We should first define a stable cycle. In a stable cycle, a small error of x0 from the exact periodic point is reduced in magnitude after n iterations, i.e., xn is closer to the exact cycle point than x0. A stable cycle is generally more meaningful than an unstable one. If we start the iteration from a random x0 it will be converged only to a stable cycle.

Bifurcation diagram

Stable cycles are often presented in a bifurcation diagram as shown below. Starting from a random x0 (0.1 here), the final xk points after a large number of iterations are plotted against the r value. Obviously, all xk points shown in the plot belong to a stable cycle.

We further note that if xk are allowed to be complex numbers, a cycle always exists, for it's always possible to find complex roots of the algebraic equation f(n)(x) = x.

Onset and bifurcation points

Our purpose is to determine the r values at which stable n-cycles exist. As we usually are interested in real r and xk, the r values for stable cycles are arranged as windows on the real axis. For example, as shown in the above figure, the stable 2-cycle exists in the window (3, 1 + √6) = (3, 3.449...). We shall call the two bounds as the onset and bifurcation points of the cycle.

Derivative condition

The onset and bifurcation points of the n-cycle can be determined from the derivative of the composite map [ f (⋯( f (x)⋯) ]′. Since a stable cycle must reduce the magnitude of x after n successive iterations, the magnitude of this derivative must be 1.0 at the two critical points. More precisely, +1 and −1 are assigned to the onset and bifurcation points, respectively. Thus by the chain rule, we have
f ′(x1) ⋯ f ′(xn) = ±1,
at the onset (+1) and bifurcation (−1) points.

The n equations, xk+1 = f (xk) (with xn = x0), along with the derivative condition, furnish n + 1 equations. Thus, we can eliminated the n variables xk and obtain the desired equation of r.

Simplified logistic map

To handle the logistic map more effectively, we change variables as (Saha and Strogatz)
x ← r (x − 1/2), R ≡ (r2 − 2 r) / 4.
such that the new map is
xk+1 = f (xk) = R − xk2.
Since f ′(x) = −2 x, the derivative condition becomes
(−2)n x1 x2 ⋯ xn = d
where d = +1 (−1) at the onset (bifurcation) points, respectively.
If we further allow complex R and xk , then we may allow d = exp(i θ) with θ running continuously from 0 to 2π. Doing so allows us to obtain continuous curves of R (instead of just two points) on the complex plane, which enclose regions of R that contain complex stable cycles. This figure is in fact the inverted Mandelbrot set. With zk = −xk and c = −R The modified logistic map becomes the quadratic map zk+1 = zk2 + c.

3-cycle

While the 1-cycle (fixed point, −1/4 < R ≤ 3/4 or 1 < r ≤ 3) and 2-cycle (3/4 < R ≤ 7/4 or 3 < r ≤ 1+√6) are generally easy to obtain (Strogatz), the 3-cycle (n = 3) is not quite obvious. The onset values R = 7/4 or r = 1 + √8 of the 3-cycle is nonetheless quite simple, and hence has been derived in several papers by different methods (see the references).

A brief and self-contained derivation of the onset and bifurcation points of the real 3-cycle is presented in the note (PDF, HTML).

As a side note, the existence of the 3-cycle in a general map is quite important, as it implies the existence of cycles of all other lengths, thus chaos, Sharkovskii and Li & Yorke.

n-cycles

For longer cycles, it's generally impossible to write down the exact critical R as radical expression. Still it is possible to identify the R value(s) as the root(s) of a polynomial with integral coefficients.

The main objective of this web page and the reference paper (Zhang, 2012) is to describe a computational method to find the polynomials. The method takes advantage of the fact that the cyclic polynomials of the n-cycle points are linearly related with one another. (An example of a cyclic polynomial is x1 x2 + x2 x3 + ⋯ + xn − 1 xn + xn x1; another example is x1 x2 ⋯ xn, which is used in the derivative condition. But note that it is not necessarily a symmetric polynomial.) Thus, we can write down a set of homogeneous linear equations regarding the cyclic polynomials. Since the cyclic polynomials are not zero altogether, the determinant of the matrix must be zero. This condition thus furnishes the desired result.

The key of the above method to establish a set of linear relations among cyclic polynomials. Note, however, the linear relations are based on the derivative condition, and are therefore different at the onset and bifurcation points. Thus, two different polynomials of R are obtained at the two points.
