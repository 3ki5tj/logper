Elementary Symbolic Dynamics by B.-L. Hao

Introduction

The page is about B.-L. Hao’s book “Elementary Symbolic Dynamics.”

Programs

Program Description
mword.py A Python script to list median words shorter than a certain length. A median word of n − 1 corresponds to a real cycle n-cycles. (The original Fortran 77 code in Chapter 3.3 of the book is heavily optimized, three-page long, and has too many GOTO statements. The shorter Python script here should be more pedagogical.)
To list all cycles up to period-7 (Table 3.1 of the book), type
python mword.py 7
renorm.ma A Mathematica script to find the universal limiting function of f(x) = 1 − A xz after many iterations by renormalization. The logistic or quadratic map corresponds to the z = 2 case. The script is quite short (about ten lines), and after adjusting parameters, such as z the number n of terms in the expansion f(x) = 1 − ∑Ak xk z. Run the program as
math < renorm.ma

Reference

  • Bailin Hao, Elementary Symbolic Dynamics and Chaos in Dissipative Systems, World Scientific, Singapore, 1989. Note. The author’s website (the above link) has a free PDF copy of the book, with several corrections.
