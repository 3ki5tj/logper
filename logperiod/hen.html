Cycles of the Hénon map

Introduction

The Hénon map is a simple two-dimensional extension of the logistic map. It is defined as:
xk+1 = 1 + yk − a xk2,
yk+1 = b xk.              
We however shall transform the variables as xk → xk / a. So the new map is
xk+1 = a + yk − xk2,
yk+1 = b xk,            
which is reduced to the logistic map when b = 0.

Programs

Program Discription
hen.ma A self-contained Mathematica script that computes the onset, bifurcation and general boundary polynomials of a and b for the n-cycles. For the onset polynomial of the 6-cycles, type
math < hen.ma 6 a
For the bifurcation polynomial of the 5-cycles, type
math < hen.ma 5 b
mkinterp.py A Python script that computes the interpolation polynomial from the list of values. This is used in parallel runs, see here for details.
